Hey Everyone,

I'm Swati Pathak, a second-year MCA student at Banasthali Vidyapith. You might wonder what's behind the scenes of my tech journey, so let me share a bit about it with you.

From BCA to MCA: The Starting Point

I started my coding adventure with BCA at Banasthali Vidyapith. Fast forward to today, and I'm knee-deep into my MCA journey. Both years have been full of challenges and discoveries.

Building Stuff: 'House Rental' Project

During my BCA days, I tackled a project called 'House Rental.' It sounds fancy, right? Well, it was my way of making a website using HTML, CSS, and PHP that could help people find houses easily. That project was like my coding playground where I learned how to bring web pages to life.

Mastering the Code: 'Library Management' Project

Now, in my MCA adventure, I took on an even cooler project named 'Library Management.' This time, I used C# and MySQL. It wasn't just about coding; it was about making a smart system to manage a library. It felt like I was crafting something really useful.

Falling in Love with Frontend: HTML and CSS Magic

You know what's my favorite part? Creating beautiful web pages! I love working on the frontend, using HTML and CSS. It's like painting a canvas but with code. I've made quite a few webpages that look good and work well, thanks to these magical languages.

Talking in Code: Java, JavaScript, C, C++

Now, let's talk about the secret languages I speak - Java, JavaScript, C, and C++. They're like the tools in my coding toolbox. Java is like my reliable friend, JavaScript adds the magic, and C/C++ are the superheroes when I need to do some heavy lifting.

Fun Fact: I'm a Continuous Learner

Beyond classrooms, I'm always on the lookout for new things to learn. Workshops, seminars, and online courses are like my treasure hunts. It's not just about getting good grades; it's about staying ahead in the exciting world of tech.

Dreaming Big: Frontend Work and More

Looking ahead, I dream of doing more cool stuff with frontend work. Making websites that not only work smoothly but also look stunning is my goal. The journey is still on, and I'm excited about the next chapters.

In a Nutshell: My Tech Story

So, that's a bit about me and my coding adventure. This blog isn't just about skills and projects; it's about my love for coding and the exciting journey I'm on. Thanks for being a part of it!
